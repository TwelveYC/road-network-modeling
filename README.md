# road-network-modeling
complex network modeling for road network modeling.

我将会使用复杂网络的模型和方法去建模路网，还会考虑到流量数据（来自滴滴盖亚的数据计划）。   
希望从这两方面去取得一些分析成果。  
可能会用到  

+ osmnx下载道路网络数据
+ https://api.openstreetmap.org/api/0.6/node/\<int:osmid>获取节点的经纬度
+ igraph和networkx用作网络分析工具
+ 高德地图和cola用作数据可视化工具
+ 等等小工具
